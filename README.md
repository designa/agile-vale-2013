Agile Vale 2013
=============

Site do evento Agile Vale 2013.

##Instruções

###Mac OS
- Instale o Node
  - `brew install node`
- Instale o grunt-cli
  - `npm install -g grunt-cli`
- Na raiz do projeto
  - `npm install`

#####Development
- Na raiz do projeto
  - `grunt watch`

#####Deploy
- Na raiz do projeto
  - `grunt deploy`
